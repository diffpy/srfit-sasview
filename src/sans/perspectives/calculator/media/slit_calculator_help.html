Description

This tool is for X-ray users to calculate the slit size (FWHM/2) for smearing based on their half beam profile data (SAXSess).

HowTo?

To calculate the slit size (FWHM/2), just load the beam profile data using the browse button.

Once a data is loaded, the slit size will be computed and show up in the text box.

Because the unit is not specified in the data file, we do not convert it into 1/A so that users are responsible for converting the unit.

Note:
This slit size calculator is working only for a beam profile data produced by 'SAXSess'.

To see the file format, check the file, 'beam profile.DAT', in the 'test' folder of SansView.
